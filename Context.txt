4. Celery worker task

Example ML worker task:

@celery.task
def process_image(image_id):

    image = load_image(image_id)

    boxes = megadetector.detect(image)

    if not boxes:
        mark_empty(image_id)
        return

    for box in boxes:

        crop = crop_box(image, box)

        species, conf = awc135.predict(crop)

        save_detection(image_id, species, conf, box)

Indexes (CRITICAL)

CREATE INDEX idx_images_camera_time
ON images(camera_id, captured_at);

CREATE INDEX idx_detections_species
ON detections(species);

Without indexes, queries become unusable at 1M+ images.
